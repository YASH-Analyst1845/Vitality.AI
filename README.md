# Vitality AI

## Project Description
Vitality AI is an AI-powered healthcare and nutrition assistant application designed to help users track health, receive personalized nutrition suggestions, and improve overall wellness.

## Features
- AI-based nutrition recommendations
- Health tracking
- User authentication
- Personalized wellness insights
- Secure data management

## Technologies Used
- Python
- Flask / React
- MySQL
